# 🥒 SauceDemo Automation - Cucumber (BDD)

This project uses **Cucumber (BDD framework)** with Selenium to automate SauceDemo website scenarios.

---

## 📌 Features

- Gherkin-based feature files
- Step Definitions in Java
- Login scenarios
- End-to-end purchase flow

---

## 🛠️ Tech Stack

- Java
- Selenium WebDriver
- Cucumber (BDD)
- Maven
- Eclipse IDE

---

## 📂 Project Structure

src/
 ├── main/java
 ├── test/java
 ├── resources (feature files)
pom.xml

---

## ▶️ How to Run

1. Run the TestRunner class
2. Ensure feature file paths are configured correctly
